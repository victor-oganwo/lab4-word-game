# Lab 4: Word Game (Hangman)

A minimal Hangman-style word guessing game for Python lab assignments.

## Requirements
- Python 3.7+
- pytest (for testing)

## How to Run
- Play the game:
  ```bash
  python main.py
  ```
- Run tests:
  ```bash
  pytest -q
  ```

## Project Files

- main.py: Main game logic, helper functions, and program entry point

- JOURNAL.md: Log of Copilot prompts, responses, and development updates

- MY_NOTES.md: Initial design notes, ideas, and selected Copilot suggestions

- REPORT.md: Reflection on the project, key learnings, and AI usage experience

## Features

- Random word selection

- Masked word display

- Tracking of guessed letters

- Win/lose detection

- Replay support

- Simple separation of logic and UI

- Tested helper functions

## Notes

This project was developed as part of a school lab assignment focused on both software development and the use of AI tools during the development process.
